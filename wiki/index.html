Joke API

Endpoint: /api/v1/joke

Method: GET

Description: Retrieves a random programming joke.

Example Response:

{
  "success": true,
  "timestamp": "2025-12-06T19:28:21.131Z",
  "meta": {
    "author": "Painsel",
    "project": "Free APIs Center",
    "github": "https://github.com/COURTESYCOIL/Free-API-Center"
  },
  "data": {
    "joke": "Why do Java developers wear glasses? Because they don't C#."
  }
}

Joke API Section

Joke API

Endpoint: /api/v1/joke

Method: GET

Description: Retrieves a random programming joke.

Example Response:

{
  "success": true,
  "timestamp": "2025-12-06T19:28:21.131Z",
  "meta": {
    "author": "Painsel",
    "project": "Free APIs Center",
    "github": "https://github.com/COURTESYCOIL/Free-API-Center"
  },
  "data": {
    "joke": "Why do Java developers wear glasses? Because they don't C#."
  }
}

Method: POST

Description: Adds a new programming joke to the collection.

Request Body:

{
  "joke": "Your new programming joke here."
}

Example Response (Success):

{
  "success": true,
  "timestamp": "2025-12-06T19:28:21.131Z",
  "meta": {
    "author": "Painsel",
    "project": "Free APIs Center",
    "github": "https://github.com/COURTESYCOIL/Free-API-Center"
  },
  "data": {
    "message": "Joke added successfully!",
    "joke": "Your new programming joke here."
  }
}

Example Response (Error):

{
  "success": false,
  "timestamp": "2025-12-06T19:28:21.131Z",
  "meta": {
    "author": "Painsel",
    "project": "Free APIs Center",
    "github": "https://github.com/COURTESYCOIL/Free-API-Center"
  },
  "data": {
    "message": "Invalid joke provided."
  }
}

Chat API Section

Chat API

The Chat API provides endpoints for managing usernames, messages, and profile pictures, enabling basic chat room functionality.

Username Endpoint

Endpoint: /api/v1/chat/username

Method: GET

Description: Retrieves a placeholder username.

Example Response:

{
  "success": true,
  "timestamp": "2025-12-06T19:28:21.131Z",
  "meta": {
    "author": "Painsel",
    "project": "Free APIs Center",
    "github": "https://github.com/COURTESYCOIL/Free-API-Center"
  },
  "data": {
    "username": "GuestUser"
  }
}

Method: POST

Description: Sets or updates a user's username.

Request Body:

{
  "username": "YourDesiredUsername"
}

Example Response (Success):

{
  "success": true,
  "timestamp": "2025-12-06T19:28:21.131Z",
  "meta": {
    "author": "Painsel",
    "project": "Free APIs Center",
    "github": "https://github.com/COURTESYCOIL/Free-API-Center"
  },
  "data": {
    "message": "Username updated to YourDesiredUsername!",
    "username": "YourDesiredUsername"
  }
}

Example Response (Error):

{
  "success": false,
  "timestamp": "2025-12-06T19:28:21.131Z",
  "meta": {
    "author": "Painsel",
    "project": "Free APIs Center",

Chat Room Integration

You can integrate a chat room into your website using these APIs. Here's how:


// Example: Fetching messages
fetch('/api/v1/chat/message')
    .then(response => response.json())
    .then(data => console.log(data));

// Example: Sending a message
fetch('/api/v1/chat/message', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify({ username: 'NewUser', message: 'Hello from my website!' }),
})
    .then(response => response.json())
    .then(data => console.log(data));
